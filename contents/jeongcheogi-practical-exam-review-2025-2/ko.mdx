## 📝 2025년 2회 정처기 실기 총평

2025년 2회차 실기 시험의 주요 특징은 다음과 같습니다.

- **😀 프로그래밍 문제 난이도 하락**: 1회차 시험의 합격률(15%)이 낮았던 점을 고려하여 프로그래밍 문제의 전반적인 난이도가 조절된 것으로 보입니다.
- **👏 이론 문제 난이도 하락** 역시 1회차 합격률이 저조했기 때문에 많은 이론 문제가 기출 문제 위주로 출제되었습니다.
- **❗️ SQL 쿼리 문제 미출제**: 많은 수험생들이 예상했던 SQL 쿼리 문제가 출제되지 않은 것이 가장 큰 특징입니다.
- **📉 C언어/Java `for`문 소멸**: 실수를 유발하기 쉬웠던 복잡한 `for`문 대신, 상대적으로 간결한 `while`문 위주로 출제되었습니다. 프로그래밍의 근본적인 지식을 평가하려는 의도로 보입니다.
- **🔥 프로그래밍 '하' 난이도 문제 소멸**: 쉽게 점수를 얻을 수 있는 '하' 난이도의 프로그래밍 문제가 사라졌습니다. 전반적인 문제 난이도를 상향 평준화하려는 출제진의 의도가 엿보입니다.

## 📋 2025년 2회 정처기 실기 이론 문제 리뷰

- SQL 쿼리 문제가 출제되지 않으면서 예상보다 DB 문제가 줄어들었습니다.
- 네트워크/OS 과목 문제가 많이 출제되었습니다. **메모리 관리, 프로세스 스케줄링 문제가 다수 출제되었습니다.**
- 25년 1회에 비해 기출 문제가 다수 출제되었습니다. **기출 문제 위주로 출제**되었기 때문에 기출 문제를 많이 풀었다면 쉽게 느껴졌을 수 있습니다.
- ❗️ 합격률이 올라갈 것으로 예상됨에 따라 25년 3회 실기 이론 문제는 난이도가 상향 조정될 것으로 보입니다.

## 📋 2025년 2회 정처기 실기 프로그래밍 문제 리뷰

- **난이도 '하'에 해당하는 문제가 없어졌습니다.** 전체적인 프로그래밍 문제 난이도를 올리겠다는 출제진의 의도가 드러났습니다.
- **앞으로 이전 기출 문제에서 난이도 '하'에 해당하는 문제를 많이 풀 필요는 없을 것 같습니다.**
- C 언어, 자바에서 'for문' 문제가 없어졌습니다. 'for문' 문제는 집중력이 좋으면 맞힐 수 있는 문제였습니다. 집중력보다는 프로그래밍 지식을 묻는 문제가 더 많이 출제될 것으로 보입니다.
