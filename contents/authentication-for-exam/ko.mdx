> 최근 보안 문제가 사회이슈로 많이 문제가 되고 있습니다.
>
> - 개인정보 유출 사건이 끊이지 않고, AI를 활용한 바이브 코딩이 보편화됨에 따라 보안 문제에 대한 관심이 높아지고 있습니다.
> - 이에 따라, 정처기 실기 문제에도 보안 문제가 주요하게 출제될 것입니다.
> - 서버 접근 제어, 인증은 실무에서도 많이 사용됩니다.
> - 정확한 인증을 사용하는 것이 네트워크 보안에 중요하기 때문에 이 부분 출제 확률 높다고 판단했습니다.

## 각 파트와 보안 관련성

| 구분               | 네트워크 공격 방어 중요성                                                                                                                                      |
| :----------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **서버 접근 통제** | **(DAC, MAC, RBAC)** 비인가자의 시스템 침입을 막고, 내부자의 권한 남용을 방지하여 중요 정보를 보호합니다.                                                      |
| **3A**             | **(인증, 권한 부여, 계정 관리)** 누가, 무엇을, 언제, 어떻게 사용했는지 기록하여 비정상적인 활동을 탐지하고, 보안 사고 발생 시 원인을 추적하는 데 필수적입니다. |
| **인증 기술**      | **(SSO, Kerberos, OAuth)** 안전한 인증 절차를 통해 공격자가 다른 사용자로 위장하는 것을 방지하고, 데이터 유출 및 시스템 장악을 막습니다.                       |
