> 정처기 실기 이론 공부를 처음부터 끝까지 다 하려는 분들을 위한 정보가 아닙니다.
>
> 다음과 같은 분들을 위한 정처기 실기 이론 공부 전략을 소개하는 글입니다.
>
> 1. 나는 정처기 실기의 넓은 범위를 모두 커버하기에는 시간이 없다.
> 2. 나는 도저히 그 넓은 범위를 공부할 수 없다.
> 3. 나는 이해하지 않으면 외우는데 자신이 없다.
> 4. 나는 코딩 문제는 어느 정도 자신이 있다.
>
> 여기에 해당되는 분들만 이 글을 읽어주시면 도움이 될 겁니다.

정처기는 총 60점 이상을 얻으면 됩니다.

20문제 중 12문제를 맞춰야 하죠.

## 최근 정처기 실기에 항상 출제되는 과목

2024년부터 2025년 1회까지 출제 문제 중 ⭐️ **항상** 출제되는 범위를 확인하면 아래와 같습니다.

| 과목               | 출제 문제수 |
| :----------------- | :---------- |
| 1과목 디자인 패턴  | 1문제       |
| 3과목 데이터베이스 | 1 ~ 2문제   |
| 6과목 자바         | 3 ~ 4문제   |
| 6과목 C언어        | 3 ~ 4문제   |
| 6과목 파이썬       | 1 ~ 2문제   |
| 7과목 SQL          | 1 ~ 2문제   |
| 9과목 보안         | 2문제       |
| 11과목 네트워크,OS | 2 ~ 3문제   |

20문제 중 17문제가 위 범위에서 ⭐️ **항상** 출제되었습니다.

2024년 2회부터는 코딩 과목(자바, C언어, 파이썬)이 9문제로 비중이 커졌습니다.

안타깝게도 코딩 과목 문제 난이도는 2025년도 1회에 대폭 상승했습니다.

### 2025년 1회 정처기 실기 코딩 과목별 문제 난이도 분석

> 1. C언어, 파이썬은 난이도 하 문제가 없었습니다.
> 2. 맞출만한 난이도 중,하 문제는 총 5문제 입니다.
> 3. 앞으로 코딩 문제에서 **최소 5문제를 맞춘다** 고 가정하고 이론 문제 전략을 보여드릴 겁니다.

| 과목 | 난이도 | 출제 문제 수 |
| :--- | :----- | :----------- |
| 자바 | 상     | 1문제        |
|      | 중     | 2문제        |
|      | 하     | 1문제        |

| 과목  | 난이도 | 출제 문제 수 |
| :---- | :----- | :----------- |
| C언어 | 상     | 2문제        |
|       | 중     | 2문제        |
|       | 하     | 0문제        |

| 과목   | 난이도 | 출제 문제 수 |
| :----- | :----- | :----------- |
| 파이썬 | 상     | 1문제        |
|        | 중     | 0문제        |
|        | 하     | 0문제        |

### 정처기 실기 이론 문제 중요성 상승

> 코딩 과목 난이도 상승으로 인해 이론에서 **7문제** 를 맞춰야 커트라인으로 합격할 수 있습니다.

아래 표는 ⭐️ **항상** 나오는 과목별 출제 예상 문제 수와 최소한으로 공부하고 합격할 우리가 각 과목당 맞춰야 할 문제 수 입니다.

| 과목               | 출제 문제수 | 맞춰야 하는 문제 수 |
| :----------------- | :---------- | :------------------ |
| 1과목 디자인 패턴  | 1문제       | 1문제               |
| 3과목 데이터베이스 | 1 ~ 2문제   | 1문제               |
| 7과목 SQL          | 1 ~ 2문제   | 1문제               |
| 9과목 보안         | 2문제       | 2문제               |
| 11과목 네트워크,OS | 2 ~ 3문제   | 2문제               |
| 총                 |             | 7문제               |

### 하지만, 보안 문제 예상이 어렵습니다.

9과목 보안은 새로운 보안 공격 용어들이 계속 나오고 있습니다. 때문에 2문제 모두 맞추는게 매우 어렵다고 봅니다.

이를 해결하기 위해 **최근 출제 경향에 따른 범위를 공부하여 1문제를 추가로 맞추는 것이 중요합니다.**

| 과목                   | 출제 문제수 |
| :--------------------- | :---------- |
| 8과목 응집도, 결합도   | 0 ~ 1 문제  |
| 10과목 테스트 커버리지 | 0 ~ 1 문제  |

## 정처기 실기 이론 문제 공부 전략

이론 7문제를 맞추기 위해 **모든 이론 범위를 공부하는 건 너무 많습니다.** 그래서 전략적으로 공부하는게 필요합니다.

프로그래밍은 언제나 그래왔듯이 트렌드가 계속 바뀝니다. 때문에 이제 사용하지 않고 의미 없는 자잘한 기술명까지 전부 외울 필요는 없습니다. 대신 중요한 개념과 중요한 기술명은 반드시 암기해야 합니다.

그간의 기출 문제를 통해 확인한 중요 세부 이론 공부 범위는 다음과 같습니다.

### 정처기 실기 이론 7문제 정답을 위한 공부 범위

> 세부 범위 클릭시 해당 범위 포스팅으로 이동합니다. 포스팅에는 개념 설명과 정처기 대비 문제가 있습니다.

| 과목                 | 맞춰야 하는 문제 수 | 세부 범위(⭐️은 무조건 나오는 범위, 🔥는 최근 출제 다수)                                                    |
| :------------------- | :------------------ | :---------------------------------------------------------------------------------------------------------- |
| 1과목 디자인패턴     | 1문제               | [⭐️디자인 패턴](/ko/blog/gof-design-pattern-cheat-sheet)                                                   |
| 3과목 데이터 베이스  | 1문제               | [🔥관계 데이터 모델](/ko/blog/relational-data-model-elements), [🔥무결성](/ko/blog/data-integrity-for-exam) |
| 7과목 SQL            | 1문제               | [⭐️SQL 명령어](/ko/blog/sql-for-exam)                                                                      |
| 9과목 보안           | 1 ~ 2문제           | [🔥공격 유형](/ko/blog/types-of-security-attack), [🔥IPSec](/ko/blog/ipsec)                                 |
| 11과목 네트워크, OS  | 2문제               | [🔥라우팅 프로토콜](/ko/blog/routing-protocol), 🔥서브넷 마스크, 🔥메모리 교체기법                          |
| 8과목 응집도, 결합도 | 0 ~ 1문제           | [🔥응집도, 결합도](/ko/blog/module-cohesion-coupling)                                                       |

## 정처기 실기 대비 책 추천

아래 수제비 책은 정처기 실기 대비에 비교적 좋은 책입니다. 블로그 포스팅과 함께 공부하면 효과적입니다.

### 수제비 정보처리기사 실기 Final 실전 모의고사

- 정보처리기사와 같은 자격증 시험은 무조건 문제를 많이 푸는 것이 도움이 됩니다.
- 굉장히 많은 문제가 수록되어 있어서 반복 숙달이 가능합니다.
- 다만, 이론 문제의 경우 너무 넓은 범위를 모두 커버하려고 해서 지엽적인 문제가 많습니다.
- 이로 인해 모의고사 점수가 안나오고 풀기 싫어질 수 있습니다.
- **틀리더라도 너무 틀린 이론 문제를 외우려고 노력하지 말고 계속 전부 다 푸는 것이 중요합니다.**
- 1회부터 풀지말고 최신 기출문제부터 풀어보는 걸 추천합니다.(역순으로 풀어보세요.)
- 1회부터 풀면 너무 틀리는 문제가 많아서 중간에 포기할 가능성이 높아보입니다.

### 수제비 정보처리기사 실기 기본서 1~2권

- 이론에 대한 설명이 잘 되어 있는 부분이 있고 아닌 부분이 있습니다.
- 설명이 잘되어 있는 부분은 그림과 함께 매우 이해하기 좋게 구성되어 있습니다.
- 단순 암기에 강한 분들이라면 이 책만 봐도 괜찮을 것 같습니다.
- 책은 기출문제를 먼저 풀고 틀린 부분을 책에서 찾아보는 방식을 권장합니다.(흥미 유발로 인한 집중력에 도움)

### 수제비 카페

- 추가적인 자료를 계속 업데이트하는데 매우 도움이 됩니다.
- [수제비 카페 - 10주 스터디 및 공부 일정 정리](https://cafe.naver.com/soojebi/232896) 따라가면 좋습니다.(모든 범위 공부에 도움이 됩니다.)
- 데일리 문제도 업데이트되는데 풀어보는 걸 권장합니다.
