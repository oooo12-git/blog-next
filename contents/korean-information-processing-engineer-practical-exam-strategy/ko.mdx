## 정처기 실기 이론 문제 주요 과목

> ❗️ SQL 쿼리(DB과목)를 이론문제에 포함한 경우

| 과목        | 출제 문제수 |
| :---------- | :---------- |
| DB          | 3 ~ 4문제   |
| SW 개발     | 2문제       |
| SW 설계     | 1 ~ 2문제   |
| 네트워크/OS | 2 ~ 3문제   |
| 보안        | 1 ~ 2문제   |
| 총          | 11문제      |

- 위 표는 2023년 부터 2025년 1회까지 주요과목 당 출제 문제수입니다.
- SQL쿼리가 포함된 DB구축 과목이 가장 중요하고 그 다음이 네트워크, OS를 포함하고 있는 프로그래밍 언어 활용 과목입니다.
- 이론 문제 구성이 2년간 안정화되면서 2025년 2회 실기에서도 위와 비슷한 구성으로 이론 문제가 출제될 것으로 예상합니다.

### 정처기 실기 이론 문제 중요성 상승

> 코딩 과목 난이도 상승으로 인해 이론에서 **7문제** 를 맞춰야 커트라인으로 합격할 수 있습니다.
>
> (2025년 1회 기준 중,하 난이도의 코딩문제는 9문제 중 5문제였습니다. **중,하 난이도 코딩 5문제를 모두 맞힌다고 가정할 경우** 이론에서 7문제를 맞춰야 합격입니다. )

| 과목        | 문제수 | 맞춰야할 문제 수 |
| :---------- | :----- | :--------------- |
| DB          | 4개    | 3개              |
| SW 개발     | 2개    | 1개              |
| SW 설계     | 2개    | 2개              |
| 네트워크/OS | 2개    | 1개              |
| 보안        | 1개    | 0개              |
| 총          | 11개   |

- 보안을 커버하려면 매우 많은 용어를 외워야 합니다.
- 보안은 중요도는 낮지만 이론 파트에서 가장 많은 시간이 소요될 것으로 예상합니다.
- SW 설계 과목은 범위가 좁고, 새로운 유형의 문제가 나오기 어려울 것으로 보입니다.
- 때문에 **보안을 버리고 SW 설계 과목을 다 맞추는 전략이 공부 시간을 줄이는데 이점이 있습니다.**

## 정처기 실기 대비 책 추천

아래 수제비 책은 정처기 실기 대비에 비교적 좋은 책입니다. 블로그 포스팅과 함께 공부하면 효과적입니다.

### 수제비 정보처리기사 실기 Final 실전 모의고사

- 정보처리기사와 같은 자격증 시험은 무조건 문제를 많이 푸는 것이 도움이 됩니다.
- 굉장히 많은 문제가 수록되어 있어서 반복 숙달이 가능합니다.
- 다만, 이론 문제의 경우 너무 넓은 범위를 모두 커버하려고 해서 지엽적인 문제가 많습니다.
- 이로 인해 모의고사 점수가 안나오고 풀기 싫어질 수 있습니다.
- **틀리더라도 너무 틀린 이론 문제를 외우려고 노력하지 말고 계속 전부 다 푸는 것이 중요합니다.**
- 1회부터 풀지말고 최신 기출문제부터 풀어보는 걸 추천합니다.(역순으로 풀어보세요.)
- 1회부터 풀면 너무 틀리는 문제가 많아서 중간에 포기할 가능성이 높아보입니다.

### 수제비 정보처리기사 실기 기본서 1~2권

- 이론에 대한 설명이 잘 되어 있는 부분이 있고 아닌 부분이 있습니다.
- 설명이 잘되어 있는 부분은 그림과 함께 매우 이해하기 좋게 구성되어 있습니다.
- 단순 암기에 강한 분들이라면 이 책만 봐도 괜찮을 것 같습니다.
- 책은 기출문제를 먼저 풀고 틀린 부분을 책에서 찾아보는 방식을 권장합니다.(흥미 유발로 인한 집중력에 도움)

### 수제비 카페

- 추가적인 자료를 계속 업데이트하는데 매우 도움이 됩니다.
- [수제비 카페 - 10주 스터디 및 공부 일정 정리](https://cafe.naver.com/soojebi/232896) 따라가면 좋습니다.(모든 범위 공부에 도움이 됩니다.)
- 데일리 문제도 업데이트되는데 풀어보는 걸 권장합니다.
