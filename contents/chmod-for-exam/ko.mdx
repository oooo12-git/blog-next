## 💡 `chmod` 명령어란?

`chmod`는 'change mode'의 약자로, 리눅스 및 유닉스 계열 운영체제에서 **파일이나 디렉토리의 접근 권한을 변경**하는 데 사용되는 명령어입니다. 정보처리기사 실기 시험에서는 특정 권한을 부여하고, 이를 8진수(Octal) 숫자로 표현하여 명령어를 완성하는 문제가 자주 출제됩니다.

## 🔑 접근 권한의 종류와 대상

`chmod`를 이해하려면 먼저 권한의 종류와 누구에게 권한을 부여할 것인지 알아야 합니다.

### 1. 권한의 종류

| 권한     | 기호 | 설명                                                    | 8진수 값 |
| :------- | :--: | :------------------------------------------------------ | :------: |
| **읽기** | `r`  | 파일 내용을 읽거나, 디렉토리 목록을 볼 수 있음          |  **4**   |
| **쓰기** | `w`  | 파일 내용을 수정/삭제하거나, 디렉토리 내 파일 생성/삭제 |  **2**   |
| **실행** | `x`  | 파일을 실행하거나, 디렉토리로 접근(cd)할 수 있음        |  **1**   |

### 2. 권한의 대상

| 대상       | 기호 | 설명                        |
| :--------- | :--: | :-------------------------- |
| **사용자** | `u`  | 파일을 소유한 사용자 (user) |
| **그룹**   | `g`  | 파일이 속한 그룹 (group)    |
| **기타**   | `o`  | 그 외 모든 사용자 (others)  |

## 🔢 8진수(Octal)로 권한 설정하기

`chmod` 명령어의 핵심은 8진수 표기법입니다. 각 권한(r, w, x)에 할당된 숫자 값을 더하여 권한 조합을 만듭니다.

- **r (읽기)** = 4
- **w (쓰기)** = 2
- **x (실행)** = 1

예를 들어, 읽고 쓸 수 있는 권한(`rw-`)은 `4 + 2 = 6`이 됩니다.

| 8진수 | 권한 조합 (`rwx`) | 설명                       |
| :---: | :---------------: | :------------------------- |
| **7** |       `rwx`       | 읽기, 쓰기, 실행 모두 가능 |
| **6** |       `rw-`       | 읽기, 쓰기 가능            |
| **5** |       `r-x`       | 읽기, 실행 가능            |
| **4** |       `r--`       | 읽기만 가능                |
| **3** |       `-wx`       | 쓰기, 실행 가능            |
| **2** |       `-w-`       | 쓰기만 가능                |
| **1** |       `--x`       | 실행만 가능                |
| **0** |       `---`       | 아무 권한 없음             |

명령어는 **`chmod [사용자권한][그룹권한][기타권한] [파일명]`** 형식으로 구성됩니다. 예를 들어 `chmod 754 file.txt`는 다음과 같은 의미입니다.

- **사용자(7)**: `rwx` (읽기+쓰기+실행)
- **그룹(5)**: `r-x` (읽기+실행)
- **기타(4)**: `r--` (읽기)

## 📝 정처기 실기 대비 문제

### 문제 1

<ProblemAnswer
  problem="리눅스 시스템에서 `report.txt` 파일의 접근 권한을 다음과 같이 변경하려고 한다. 8진수 표기법을 사용하여 `chmod` 명령어를 완성하시오.

- 사용자: 읽기, 쓰기 권한
- 그룹: 읽기 권한
- 기타: 아무 권한 없음"
  correctAnswer="chmod 640 report.txt"
  />

<Details summary="문제 풀이">

각 대상의 권한을 8진수 숫자로 변환합니다.

1.  **사용자**: 읽기(4) + 쓰기(2) = **6**
2.  **그룹**: 읽기(4) = **4**
3.  **기타**: 아무 권한 없음 = **0**

세 숫자를 조합하면 `640`이 됩니다. 따라서 전체 명령어는 다음과 같습니다.

```bash
chmod 640 report.txt
```

</Details>

### 문제 2

<ProblemAnswer
  problem="`script.sh`라는 셸 스크립트 파일이 있다. 이 파일의 사용자는 모든 권한을 가져야 하고, 그룹 및 이 외는 읽고 실행할 수만 있도록 권한을 설정하는 `chmod` 명령어를 작성하시오."
  correctAnswer="chmod 755 script.sh"
/>

<Details summary="문제 풀이">

각 대상의 권한을 8진수 숫자로 변환합니다.

1.  **사용자**: 읽기(4) + 쓰기(2) + 실행(1) = **7**
2.  **그룹**: 읽기(4) + 실행(1) = **5**
3.  **기타**: 읽기(4) + 실행(1) = **5**

세 숫자를 조합하면 `755`가 됩니다. 따라서 전체 명령어는 다음과 같습니다.

```bash
chmod 755 script.sh
```

</Details>
