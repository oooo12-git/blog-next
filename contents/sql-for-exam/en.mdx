
```sql
DELETE FROM table_name
```

```sql
UPDATE table_name

WHERE condition;
```

## 🗑️ When removing a table

### If the table is not referenced, it's deleted; if it is referenced, deletion fails

```sql
DROP TABLE table_name RESTRICT;
```

<ProblemAnswer
  problem="Write a DDL statement to delete the Employee table only if it is not referenced by another table."
  correctAnswer="DROP TABLE Employee RESTRICT;"
/>

### Deletes the table and any referencing tables in a cascade

```sql
DROP TABLE table_name CASCADE;
```

<ProblemAnswer
  problem="Fill in the parentheses with the correct option.
  You are trying to DROP a table that has a foreign key constraint. To remove the table and all referencing tables in a cascade, use the ( 1 ) option. To prevent deletion if the table is being referenced, use the ( 2 ) option."
  correctAnswer="1 : CASCADE, 2 : RESTRICT"
/>

<ProblemAnswer
  problem="Write a DDL statement to remove the Company table and all data that references it."
  correctAnswer="DROP TABLE Company CASCADE;"
/>

## 🔑 Revoking permissions

```sql
REVOKE privilege_type ON TABLE_NAME FROM user_name;
```

<ProblemAnswer
  problem="Write an SQL statement for an administrator to revoke the UPDATE permission on the Employee table from a USER."
  correctAnswer="REVOKE UPDATE ON Employee FROM USER;"
/>

## 📉 Sorting two columns in descending order

> To sort by column1 in descending order first, and then by column2 in descending order...

```sql
SELECT column1, column2, ...
FROM table_name
ORDER BY column1 DESC, column2 DESC;
```

<ProblemAnswer
  problem="Write an SQL statement to sort the data in the Company table by DepartmentLocation and DepartmentName in descending order.
  (Sort by DepartmentLocation descending first, and if locations are the same, then by DepartmentName descending)
  (Select all columns)"
  correctAnswer="SELECT * 
  FROM Company 
  ORDER BY DepartmentLocation DESC, DepartmentName DESC;"
/>

## 📇 Creating an index

```sql
CREATE INDEX index_name ON table_name (column1, column2, ...);
```

<ProblemAnswer
  problem="Write an SQL statement to create an index named 'DepartmentIndex' on the DepartmentName and DepartmentLocation columns of the Company table."
  correctAnswer="CREATE INDEX DepartmentIndex ON Company (DepartmentName, DepartmentLocation);"
/>

## 🎯 Querying data with values within a specific range

### BETWEEN

```sql
SELECT column1, column2, ...
FROM table_name
WHERE column1 BETWEEN value1 AND value2;
```

<ProblemAnswer
  problem="Write an SQL statement to query departments from the Company table where the DepartmentID is between 1 and 3.
  (Select all columns)"
  correctAnswer="SELECT * FROM Company WHERE DepartmentID BETWEEN 1 AND 3;"
/>

### IN

```sql
SELECT column1, column2, ...
FROM table_name
WHERE column1 IN (value1, value2, ...);
```

<ProblemAnswer
  problem="Write an SQL statement to query departments from the Company table where the DepartmentName is 'Development', 'HR', or 'General Affairs'.
  (Select all columns)"
  correctAnswer="SELECT * FROM Company WHERE DepartmentName IN ('Development', 'HR', 'General Affairs');"
/>

## 🔍 Querying data that starts with a specific character

```sql
SELECT column1, column2, ...
FROM table_name
WHERE column1 LIKE 'value%';
```

<ProblemAnswer
  problem="Write an SQL statement to query departments from the Company table that start with 'Dev'.
  (Select all columns)"
  correctAnswer="SELECT * 
  FROM Company 
  WHERE DepartmentName LIKE 'Dev%';"
/>

## 🤔 Querying data with exactly one character before a specific character

```sql
SELECT column1, column2, ...
FROM table_name
WHERE column1 LIKE '_value';
```

<ProblemAnswer
  problem="Write an SQL statement to query departments from the Company table where the name has exactly one character before 'Dept'.
  (Select all columns)"
  correctAnswer="SELECT * 
  FROM Company 
  WHERE DepartmentName LIKE '_Dept';"
/>

## 📊 Querying with a condition on the sum

```sql
SELECT column2, SUM(column1) AS total
FROM table_name
WHERE condition;
GROUP BY column2;
HAVING SUM(column1) condition;
```

<ProblemAnswer
  problem="Write an SQL statement to query the department name and the total number of employees for departments with 10 or more employees.
  (Company table columns: DepartmentName, NumberOfEmployees)"
  correctAnswer="SELECT DepartmentName, SUM(NumberOfEmployees) AS TotalEmployees 
  FROM Company 
  GROUP BY DepartmentName 
  HAVING SUM(NumberOfEmployees) >= 10;"
/>

## ❓ Querying NULL value data

> Used to find records where a specific column's value is NULL.

```sql
SELECT column1, column2, ...
FROM table_name
WHERE column_name IS NULL;
```

<ProblemAnswer
  problem="Write an SQL statement to query the names of employees from the 'Employee' table who have not yet been assigned a department (DepartmentCode is NULL).
  (Employee table columns: Name, DepartmentCode)"
  correctAnswer="SELECT Name 
  FROM Employee 
  WHERE DepartmentCode IS NULL;"
/>

## 🏗️ Altering table structure

> Used to add a new column to an existing table.

```sql
ALTER TABLE table_name
ADD column_name datatype;
```

<ProblemAnswer
  problem="Write an SQL statement to add a 'Contact' column of type VARCHAR(20) to the 'Employee' table."
  correctAnswer="ALTER TABLE Employee ADD Contact VARCHAR(20);"
/>

## 🔄 Using a subquery

> A query nested inside another SQL statement, useful for handling complex conditions.

```sql
SELECT column1, column2, ...
FROM table_name
WHERE column_name IN (SELECT column_name FROM another_table WHERE condition);
```

<ProblemAnswer
  problem="Write an SQL statement using a subquery to retrieve the names of all employees from the 'Employee' table who work in a department located in 'Seoul'.
  (Department table columns: DepartmentCode, DepartmentName, Location)
  (Employee table columns: EmployeeID, Name, DepartmentCode)"
  correctAnswer="SELECT Name 
  FROM Employee
  WHERE DepartmentCode IN (SELECT DepartmentCode FROM Department WHERE Location = 'Seoul');"
/>

## 🔢 Counting the number of data

> Used to count the number of rows that satisfy a specific condition.

```sql
SELECT COUNT(column_name)
FROM table_name
WHERE condition;
```

<ProblemAnswer
  problem="Write an SQL statement to calculate the total number of employees in the 'Development' department from the 'Employee' table."
  correctAnswer="SELECT COUNT(*) FROM Employee WHERE Department = 'Development';"
/>
